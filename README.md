# Avatar Couture

Vite + React fashion design prototype with a live outfit studio, avatar preview, curated collections, and AI stylist chat. Entire website created by Orbit Studios.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
